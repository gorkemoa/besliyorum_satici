# Theme Standards

This document outlines the standard colors and typography used in the Besliyorum Satıcı application.

## Colors

### Core Palette
- **Primary Color**: `#E62F35` (Red) - Used for primary actions, buttons, and branding.
- **Secondary Color**: `#FFFFFF` (White) - Used for backgrounds and text on primary color.
- **Error Color**: `#D32F2F` (Deep Red) - Used for error inputs.
- **Success Color**: `#2E7D32` (Green) - Used for confirmation actions and success states.

### Semantic Colors for Dialogs
- **Dialog Title**: `#B71C1C` (Dark Red)
- **Dialog Text**: `#B71C1C` (Dark Red)
- **Cancel Button**: `#C62828` (Red)
- **Confirm Button**: `#2E7D32` (Green)

## Typography

The application uses **Poppins** font family.

- **Headlines**: Bold, Dark Text
- **Body**: Regular, Dark Text (or Grey for secondary info)
- **Button Text**: Medium/Semi-Bold, White

## Usage Guide

Use `AppTheme.lightTheme` to access the theme data.
Use `AppTheme.successColor` for success states not covered by the default theme.
